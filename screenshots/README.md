# Screenshots Directory

This directory contains screenshots of the FakeStore Shop application.

## Required Screenshots

Please add the following screenshots to this directory:

1. **home-page.png** - Screenshot of the home/welcome page
2. **products-page.png** - Screenshot of the products catalog page with grid layout
3. **list-page.png** - Screenshot of the list view page
4. **detail-page.png** - Screenshot of a product detail page
5. **dark-mode.png** - Screenshot showing the dark mode theme

## How to Take Screenshots

1. Start the development server:
   ```bash
   npm run dev
   ```

2. Open your browser and navigate to `http://localhost:5173`

3. Take screenshots of each page:
   - Home page: `http://localhost:5173/`
   - Products page: `http://localhost:5173/products`
   - List page: `http://localhost:5173/list`
   - Detail page: `http://localhost:5173/products/1` (or any product ID)
   - Dark mode: Toggle dark mode and take a screenshot of any page

4. Save the screenshots with the exact filenames listed above in this directory.

## Screenshot Tips

- Use full-page screenshots when possible
- Ensure the browser window is at a reasonable size (1920x1080 or similar)
- Make sure all content is loaded before taking screenshots
- For dark mode, toggle the theme using the button in the navbar
